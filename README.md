# Line TODO Management Bot
## Overview
- Add your todo task.
- Delete your completed task if you type the task's name.
- Delete all your task if you type "クリア" or "clear"

### Techs
- Python 3.8.1
- Flask
- xserver (Hosting server)
- mysql
